server_data.txt:
url, auth, headers - информация для совершения запроса кк серверу, закомменченные данные, это ссылка на другую базу данных от Максима, не знаю точно какая именно тебе нужна, сам выбирай.
TIME_LIMIT = предел считывания информации в секундах (для графика)
TIME_INTERVAL = интервал запросов в милисекундах

requirements.txt библиотеки, возможно будет ругаться на plt.show(), тогда надо прописать sudo apt install python3-tk

main.py
get_apparat_data - получение данных
create_plot - вывод графика во вреемени


запуск:
python3 main.py get_data - получение названий элементов, которые есть в базе данных
python3 main.py get_data <имя элемента> - запись значения элемента в файл каждую секунду, название файла <имя элемента>.txt 
python3 main.py create_plot <имя элемента> - вывод графика и дозапись данных в файл, с интервалом = TIME_INTERVAL
